## ⚡ Overview

**ComplianceGuard AI (Policy Compliance Checker Agent)** protects organizations by allowing direct audits between unstructured textual content against rigid corporate policy rules. It actively identifies breaches, extracts severity matrices, and issues documented remediation guidance.

## 🚀 Key Features

*   **🛡️ Liability Shield:** Ensures human resources, emails, and outward content does not violate data-privacy laws or internal rulebooks.
*   **📊 Severity Logic:** Not all rule breakages carry the same weight. Evaluates boundaries directly, producing `Low/Medium/High/Critical` threat tags dynamically.
*   **💠 Red-Team UI:** A distinct Crimson & Slate Dark Mode Dashboard built to immediately highlight and contrast policy infringement risks.
*   **🧠 Total Orchestration:** Works over Litellm with any model you need—from `Claude-3.5-Sonnet` for writing elegance to `Llama 3` for local privacy when analyzing extremely sensitive internal documents.

## ⚙️ Quick Start

### 1. Execute CLI Terminal Audit
Inject rules into `policy.txt` and check against `content.txt`:
```bash
cd "HR , Legal & Compliance Agents/Policy compliance checker agent"
python agent.py
```

### 2. Enter Visual Interface
```bash
streamlit run app.py
```
*(Available locally on default port 8501-8512).*
