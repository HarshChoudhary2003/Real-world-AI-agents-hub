# Web Research Agent 🌐🔍

An enterprise-grade autonomous intelligence gathering agent built to comprehensively synthesize complex informational topics, market trends, and industry updates into clear, actionable executive insights.

## Features ✨

- **Autonomous Intelligence Gathering:** Connects seamlessly with OpenAI's most advanced LLMs (GPT-4o, GPT-4 Turbo) to dynamically evaluate topics based on your specified constraints.
- **Enterprise UI Ecosystem:** A sleek, fully dark-themed, responsive dashboard built on Streamlit with custom advanced CSS styling.
- **Structured Synthesis:** Ensures intelligence is returned as rigorously parsed JSON, separating:
  - High-level Executive Summaries
  - Actionable Key Findings
  - Dynamic Market Trends
  - Critical Open Questions
- **One-Click Export:** Immediately export reports as Raw JSON or nicely formatted Text Briefings for executive teams.
- **Parametric Controls:** Easily adjust agent parameters directly from the dashboard—tweak agent 'Creativity' (temperature) and LLM architecture.

## Installation 🔧

1. **Clone the repository / Navigate to the directory:**
   ```bash
   cd "Research & Analysis Agents/Web research agent"
   ```

2. **(Optional but Recommended) Create a Virtual Environment:**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use: venv\Scripts\activate
   ```

3. **Install Dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

## Usage 🚀

1. **Launch the Dashboard:**
   ```bash
   streamlit run app.py
   ```

2. **Configure the Agent:**
   - On the left sidebar, enter your **OpenAI API Key**.
   - Select your preferred **LLM Architecture** (e.g., `gpt-4o`).
   - Adjust the **Creativity (Temperature)** variable (0.2 - 0.4 recommended for factual research).

3. **Deploy Research:**
   - Define your **Objective & Scope** in the main window (Topic, Scope, Focus Areas, Timeframe).
   - Click **Deploy Research Agent 🚀**.

4. **Review & Export:**
   - Navigate through the *Executive Summary*, *Detailed Analysis*, and *Raw Data* tabs.
   - Download the findings for your team.

## Project Structure 📁

- `app.py`: The main enterprise Streamlit application, housing both the UI logic and agent communication components.
- `agent.py`: A lightweight core CLI version for developers wanting to run the agent in headless mode.
- `requirements.txt`: Package dependencies.
- `input.txt` / `research.json` / `research.txt`: Examples of raw inputs and structured outputs generated by the agent.
