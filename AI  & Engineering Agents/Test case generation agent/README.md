## ⚡ Overview

**TestForge AI** deterministically analyzes python module blocks and functions to synthesize comprehensive regression suites spanning expected cases, logical boundary conditions, and exception routing parameters—without hallucinating behavior.

## 🚀 Key Features

*   **🛡️ Edge Case Simulations:** Pushes beyond standard assertions and checks bounds limits intrinsically.
*   **🧠 Intelligence Routing:** Progressive Provider ➡️ Engine ➡️ Token Key dropdown structure utilizing LiteLLM orchestration. 
*   **🧩 `unittest` Compilation:** Auto-generates fully functional Python Unittest structures ready to test target modules.
*   **📊 Test Analytics:** Outputs detailed report definitions detailing assertion rules locally in JSON.

## ⚙️ Quick Start

### 1. Execution

To utilize the core CLI parser functionality:
```bash
python agent.py
```

It reads `source_code.py` and emits standard `generated_tests.py` & `test_cases_report.json`.

### 2. Enter the Dashboard

```bash
streamlit run app.py
```
Experience the Elite Yellow Dark-Mode UI and simulate suites directly from browser inputs without spinning up terminal configs.
