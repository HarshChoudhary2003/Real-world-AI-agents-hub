# 🧠 TaskBrain AI | Enterprise Edition

**TaskBrain AI** is a state-of-the-art Daily Task Prioritization Agent. Designed to transform how professionals and teams plan their day, it uses algorithmic scoring combined with cutting-edge OpenAI integration to organize, optimize, and execute work efficiently.

Whether you're juggling deadlines, roadblocked tasks, or multiple projects, TaskBrain AI dynamically assesses your bandwidth and mathematically generates the perfect daily operational plan.

---

## ✨ Core Features

### 1. 📊 Commander Dashboard
An ultra-premium, interactive dashboard built with **Streamlit** that visualizes your day.
- **Dynamic Prioritization:** Sorts your raw tasks into *Alpha Priority (Top 3)*, *Bravo (Next 5)*, *Roadblocks*, and *Deferred/Backlog* tasks automatically.
- **Real-Time Bandwidth Tracking:** Automatically tracks your remaining time quota (e.g., 120 minutes per day) and defers tasks that push you over the limit.
- **One-Click Exports:** Download your daily tactical plan into raw JSON or a neatly formatted text briefing.

### 2. 🤖 Artificial Intelligence Integrations
*Unlock premium functionality by entering an OpenAI API Key in the side panel:*
- **🗣️ Natural Language Ingestion:** No more manual form entry. Speak or type unformatted text (e.g., *"Remind me to draft that email to Sarah tomorrow, takes 20 mins, extremely important"*), and the AI will extract the title, duration, and impact, converting it instantly into system data.
- **🌅 Executive Morning Briefing:** Get a contextual, motivational summary of your day generated by AI. Choose different assistant "personalities" (e.g., *Executive Coach*, *Drill Sergeant*, or *Zen Master*).
- **📋 Sub-task Generation Engine:** Feeling overwhelmed by a complex task? One click instantly asks the AI to break the task down into 3-4 highly actionable sub-steps.

### 3. 📈 Interactive Analytics & Insights
Integrated with **Plotly** to visually dissect your workload:
- **Effort vs. Urgency Matrix:** See exactly which low-effort, high-impact tasks are "Quick Wins".
- **Impact Level Distributions:** Visualize how much of your day is spent on high vs. low impact items.

## ⚙️ How the Algorithms Work (The Brains)

At the heart of the engine is `agent.py`, a complex ruleset that grades every task based on these configurable metrics:

1. **Urgency Score:** Exponential curve scoring based on days until the deadline (Overdue/Today = 5 points, Due in 7 days = 2 points, No deadline = 0.5 points).
2. **Impact Score:** Strict Low (1), Medium (2), High (3) multiplier.
3. **Quick-win Bonus:** Tasks taking <= 15 minutes get an automatic boost so quick wins don't get buried.
4. **Blocked Penalty:** Tasks marked as "Blocked" are severely penalized and routed to the "Roadblocks" section, ensuring they don't clog up your operational pipeline.

The scoring algorithm is: 
`(Urgency * Urgency_Weight) + (Impact * Importance_Weight) + Quick_Win_Bonus - Blocked_Penalty`

## 🚀 Installation & Setup

### Requirements
- Python 3.9+
- `pip` or standard package manager

### 1. Install Dependencies
Run the following command to install the required libraries:
```bash
pip install streamlit pandas plotly openai
```

### 2. Prepare your Tasks (Optional)
The system reads a local `tasks.csv` file by default (or you can upload one via the UI). To use the local file, ensure it matches this schema:

| title | description | deadline | effort | impact | blocked | tags |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| Send proposal | Finalize and email | 2025-12-22 | 25m | high | no | work |
| Fix login bug | Patch issue | 2025-12-24 | 15m | high | no | dev,work |

*(Note: The system is robust. Effort can be written as `15m`, `15min`, `15`, or even `S/M/L` sizes!)*

### 3. Run the Application
Start the powerful Streamlit engine from your terminal:
```bash
streamlit run app.py
```
*Note: The system will automatically open a local server at `http://localhost:8501`.*
