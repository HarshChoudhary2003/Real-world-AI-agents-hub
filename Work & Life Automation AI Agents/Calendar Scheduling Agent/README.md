# 📅 AI Calendar Scheduler (ProductivityOS)

The AI Calendar Scheduler is a high-precision temporal architect. It goes beyond simple task lists to build a high-density daily schedule optimized for the user's biological peak focus windows and energy-depletion curve.

## 🧠 Core Intelligence
- **Energy-Based Planning**: Matches high-intensity tasks (Deep Work) with the user's peak focus windows.
- **Chronotype Matching**: Adjusts scheduling based on "Morning Focus", "Balanced", or "Evening Focus" preferences.
- **Buffer Node Insertion**: Automatically inserts buffer blocks to prevent burn-out and decision fatigue.
- **Precision Time Blocking**: Converts a raw list of tasks into a surgical sequence of time-allocated execution nodes.

## 🚀 Key Features
- **🔥 Premium Glassmorphic UI**: Ultra-modern, dark-themed dashboard for high-performers.
- **📅 Visual Timeline Sequence**: A clear, color-coded execution sequence for the entire day.
- **📊 Energy Tracking Score**: AI-calculated efficiency score for the generated schedule.
- **📥 One-Click Export (.ics)**: Download your schedule and sync it with Google Calendar, Apple Calendar, or Outlook.
- **💡 Strategic Productivity Tips**: Person-specific advice for maintaining focus throughout the day.
- **🧬 Model Agnostic (LiteLLM)**: Fully compatible with GPT-4, Gemini Pro, Claude 3, and Llama 3 models.

## 🛠 Setup & Launch
1. Clone the repository and navigate to this folder.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   streamlit run app.py
   ```
