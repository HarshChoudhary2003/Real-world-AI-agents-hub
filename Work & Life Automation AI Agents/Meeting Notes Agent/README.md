# 📝 Meeting Intelligence Agent (ProductivityOS)

The Meeting Intelligence Agent is a tactical deconstruction engine designed to transform messy, unstructured temporal logs (meeting transcripts) into high-fidelity actionable intelligence.

## 🧠 Core Intelligence
- **Narrative Neural Synthesis**: Generates a strategic summary that captures the core intent and "vibe" of the meeting.
- **Surgical Action Node Extraction**: Identifiers specific, owner-attributed tasks with priority mapping and deadline detection.
- **Strategic Decision Mapping**: Highlights the hard choices made during the meeting to prevent circular discussions.
- **Sentiment & Vibe Audit**: Analyzes the meeting's emotional landscape (Positive, Tense, Neutral) for better context.
- **Future Agenda Architecting**: Automatically generates a follow-up agenda based on unresolved questions.

## 🚀 Key Features
- **🔥 Premium Glassmorphic UI**: Ultra-modern, dark-themed dashboard with tactical-grid layout.
- **📁 Multi-Format Ingestion**: Supports direct text pasting or file uploads (.txt, .md).
- **✅ Action Registry UI**: A clean, prioritized view of "Who does What by When."
- **📊 Export To Brief**: Download meeting intelligence as a clean, standardized text brief for distribution.
- **🧬 Model Agnostic (LiteLLM)**: Fully compatible with GPT-4, Gemini Pro, Claude 3.5, and Llama 3 models.

## 🛠 Setup & Launch
1. Clone the repository and navigate to this folder.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   streamlit run app.py
   ```
