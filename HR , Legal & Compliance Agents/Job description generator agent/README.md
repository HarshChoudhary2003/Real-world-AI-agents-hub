## ⚡ Overview

**TalentForge AI (Job Description Generator)** takes basic bullet-point requirements, constraints, and locations from hiring managers and expands them into professionally structured, inclusive, and standardized job descriptions. It outputs data structurally ready for Applicant Tracking Systems (JSON) or Human Resources publication (TXT).

## 🚀 Key Features

*   **🛡️ Bias Eradication:** Neural prompt constraints automatically prevent exclusionary or hyper-aggressive syntax, ensuring inclusive language parsing.
*   **📊 Structured Outputs:** Generates perfectly aligned generic JSON keys representing `Required Qualifications`, `Preferred`, `Responsibilities`, and `Benefits`.
*   **💠 Executive HR Dashboard:** Features a stunning Midnight Blue & Neon Cyan visual interface configured for instant copy/pasting payload execution.
*   **🧠 Agnostic Backend:** Switch seamlessly between OpenAI, Anthropic, DeepSeek, or localized models via Ollama. 

## ⚙️ Quick Start

### 1. Terminal Operations
Pass your brief to `role.txt` and execute:
```bash
cd "HR , Legal & Compliance Agents/Job description generator agent"
python agent.py
```

### 2. Streamlit Dashboard
Launch the graphical talent engine interface:
```bash
streamlit run app.py
```
