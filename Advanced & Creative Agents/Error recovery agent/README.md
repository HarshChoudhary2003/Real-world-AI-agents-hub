**HealDesk.AI** is an autonomous anomaly detection and self-healing engine designed to restore system stability without operator intervention. 

### 🚀 Key Features
- **Anomaly Architecture Scan**: Multi-level classification of system errors (DB Timeouts, Auth Failures, etc.).
- **Autonomous Recovery Timeline**: Visual step-by-step history of all maneuvers taken.
- **Escalation Module**: Triggers "Red Alert" (Gold/Red UI) for scenarios requiring manual intervention.
- **Outcome KPI Dashboard**: Strategic metrics showing Recovered vs. Degraded vs. Failed statuses.

### 🛠️ Tech Stack
- **AI**: GPT-4o-mini (Self-correction reasoning).
- **Core**: Python 3.11+, Streamlit, `json`.

### ⚡ Quick Start
**To run the HealDesk Studio:**
```bash
streamlit run app.py
```
