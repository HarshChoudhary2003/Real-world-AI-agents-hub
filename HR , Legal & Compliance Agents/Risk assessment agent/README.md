## ⚡ Overview

**RiskForge AI (Risk Assessment Agent)** allows organizations to pass in raw operational plans, product launches, or system architectures and immediately receive a highly robust, enterprise-grade Risk Assessment matrix. The system automatically categorizes identified threats by Likelihood and Impact, allowing management to see the true risk.

## 🚀 Key Features

*   **🛡️ Multi-Axis Threat Modeling:** The agent identifies implicit risks (like GDPR infractions resulting from EU tracking) that humans might overlook.
*   **📊 Structured Mitigation Logging:** Not only identifies failure points but provides direct implementation steps to shield against those specific attacks or legal breaches.
*   **💠 Executive War Room UI:** An Amber & Charcoal dark mode interface designed specifically to handle high-stress data aggregation effortlessly.
*   **🧠 Total Orchestration:** Works over Litellm with any model you need—from `Claude-3.5-Sonnet` for writing elegance to `Llama 3` for fully localized, air-gapped system scanning.

## ⚙️ Quick Start

### 1. Execute CLI Pipeline
Inject the operational scenario into `scenario.txt` and execute:
```bash
cd "HR , Legal & Compliance Agents/Risk assessment agent"
python agent.py
```

### 2. Enter Visual Interface
```bash
streamlit run app.py
```
*(Available locally on default port 8501-8515).*
