# ⚖️ Decision Intelligence Agent (ProductivityOS)

The Decision Intelligence Agent is a high-fidelity strategic deconstruction engine designed to eliminate decision-fatigue and cognitive bias. It transforms complex dilemmas into logical choice architectures for rapid, confident execution.

## 🧠 Core Intelligence
- **Multi-Variant Choice Analysis**: Deconstructs every potential option into surgically precise Pros and Cons, weighted against user priorities.
- **Risk-Reward Audit Nodes**: Identifies hidden risks, second-order effects, and strategic importance scores for every choice pathway.
- **Confidence Synthesis Model**: Provides a logic-driven confidence metric (0-10) combined with ambiguity detection for unknown variables.
- **Reasoning Architecture Deconstruction**: Generates a high-precision narrative explaining the logical synthesis of the "Best Choice."
- **Strategic Mentor-Style Coaching**: Offers a "Decision Pith" node (high-impact advice) tailored to the complexity of the dilemma.

## 🚀 Key Features
- **🔥 Premium Glassmorphic UI**: High-density dark-themed dashboard with tactical-grid and comparison expanders.
- **📊 Metric Consistency Hub**: Visual indicators for logic confidence scores and execution risk levels.
- **🛡️ Choice Comparison Grid**: Side-by-side analysis of various options with color-coded success markers.
- **🧪 Ambiguity Detection**: AI-led identification of incomplete data or high-variance nodes in the decision process.
- **⬇️ Decision Memo Export**: One-click download as a clean, standardized strategic brief for your records.

## 🛠 Setup & Launch
1. Clone the repository and navigate to this folder.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   streamlit run app.py
   ```
