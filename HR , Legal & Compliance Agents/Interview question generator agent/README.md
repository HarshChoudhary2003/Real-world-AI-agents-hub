## ⚡ Overview

**QuestForge AI (Interview Question Generator Agent)** completely automates the screening prep phase. By providing basic inputs like stack requirements and seniority levels, the agent orchestrates arrays of Behavioral, Technical, and System-Scenario questions that avoid illegal HR queries and focus exclusively on core competency extraction.

## 🚀 Key Features

*   **🛡️ Bias Filtering:** Trained specifically to exclude legally compromising lines of questioning.
*   **📊 Vector Scaling:** Adjusts questioning complexity relative to the 'Seniority' variable directly.
*   **💠 Executive Interview Dashboard:** A brilliant Amber & Obsidian UI built specifically for rapidly cycling applicant structures before meetings.
*   **🧠 Provider Agnostic:** Progressive input structures allow scaling to enterprise local boundaries via Ollama/Grok/Antrhopic integrations.

## ⚙️ Quick Start

### 1. Headless Generation
Place your requisition parameters inside `role.txt` and launch:
```bash
cd "HR , Legal & Compliance Agents/Interview question generator agent"
python agent.py
```

### 2. Enter Visual Auditing
Access the graphical interview console:
```bash
streamlit run app.py
```
