**Ethics.AI** is a high-fidelity assessment and compliance guardrail engine designed to sanitize proposed maneuvers in mission-critical autonomous workflows. 

### 🚀 Key Features
- **Neural Framework Audit**: Evaluates actions against defined `policies.txt` (irreversible impact, PII disclosure, etc.).
- **Decision Matrix Visualizer**: Large color-coded indicators (ALLOW, BLOCK, ESCALATE).
- **Immutable Audit Vault**: Detailed rationale and notes recorded in `ethics_output.json`.
- **Governance Strategic Panel**: Focused workspace for high-stakes compliance triage.

### 🛠️ Tech Stack
- **AI**: GPT-4o-mini (Ethical reasoning).
- **Core**: Python 3.11+, Streamlit, `json`.

### ⚡ Quick Start
**To run the Governance Studio:**
```bash
streamlit run app.py
```
