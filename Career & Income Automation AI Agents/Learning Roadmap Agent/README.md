# 📚 Learning Roadmap Agent

The **Learning Roadmap Agent** is a strategic career architect in the **CareerOS** suite, designed to transform a candidate's current baseline into mastery for any target role.

## 🚀 Core Intelligence: The 90-Day Blueprint
This agent performs a deep-dive comparison and generates:
1.  **30/60/90 Day Plan**: Clear foundational, intermediate, and advanced milestones.
2.  **Weekly Breakdown**: Minute focus areas and specific tasks for every week of the journey.
3.  **Portfolio Project Pipeline**: Hands-on, industry-standard projects to demonstrate impact.
4.  **Strategic Overview**: High-level transformation strategy tailored specifically to the candidate's baseline.

## 🛠️ Features
- **Trajectory Mapping**: Calculates the most efficient path to master mandatory technical nodes.
- **Portfolio-Focus**: Prioritizes projects that make you stand out to hiring managers.
- **Multi-Model Intelligence**: Switch between OpenAI Core, Gemini Pro, or Groq for varying instructional depth.
- **Glassmorphic UI**: Premium high-density dashboard with timeline views and resource sidebars.
