# 💡 LinkedIn Viral Post Architect & Ideation Agent

Transform a single thought into a week of high-converting, psychologically optimized LinkedIn content. 

This is not a rudimentary prompt generator. The **Elite LinkedIn Viral Architect** is a fully automated content strategizing pipeline. It systematically parses your core "Personal Angle" into five distinct, ready-to-publish posts using proven engagement frameworks like AIDA (Attention-Interest-Desire-Action) and PAS (Problem-Agitate-Solve).

## 🚀 Why This is an Industry-Grade Tool

* **Multi-LLM Omnichannel Pipeline:** Programmed to support not just OpenAI, but Anthropic’s Claude 3.5 Sonnet (the current reigning champion for natural voice copying) and Google's Gemini 2.0 Flash/Pro integrations. 
* **Psychological Hooks:** Automatically drafts attention-grabbing, scroll-stopping first sentences specifically formatted for LinkedIn's "See More" cut-off boundary.
* **Full Draft Architect:** No more "ideas" that you still have to hand-write. This Agent outputs the *Actual Final Draft* of the post, complete with perfect paragraph spacing and targeted emojis to stop the feed cycle.
* **Algorithmic Asset Mapping:** Recommends the perfect visual (carousel, high-contrast photo, meme, or text-only) mapped to your generated text, while embedding the best 3-5 hashtags to game organic reach.
* **Pre-Computation Engagement Scores:** Evaluates its own drafts before giving them to you, delivering an estimated `94/100` Engagement Potential metric to prioritize publishing order.

## 🛠 Setup & Installation

The **Elite Architect** is portable and natively connects into CLI workflows or Visual interfaces with a click.

**1. Install Core Requirements:**
```bash
pip install openai anthropic google-generativeai streamlit
```

**2. Start the Agent Dashboard (Recommended):**
Skip terminal keys entirely and manage inputs safely:
```bash
cd "LinkedIn post ideation agent"
python -m streamlit run app.py
```
*(Your control panel will host beautifully at http://localhost:8502)*

**3. Headless / CLI Operation:**
Provide an `input.txt` inside the root directory and configure an environment key:
```bash
$env:OPENAI_API_KEY="sk-..."  # or ANTHROPIC_API_KEY / GEMINI_API_KEY
python agent.py
```
The raw `ideas.json` (perfect for programmatic pipelines) and human-readable `ideas.txt` are populated instantaneously.
