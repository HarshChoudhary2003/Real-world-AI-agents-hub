# 💰 CapitalMind: Investment Thesis Generator Agent

**CapitalMind** is a high-performance AI agent designed to architect rigorous, evidence-driven investment theses. It goes beyond simple analysis by mapping core rationale, identifying specific catalysts, and surfacing realistic risks, helping investors build conviction based on first-principles logic rather than hype.

---

## 🌟 Premium Features

- **🏛 Rigorous Thesis Architecture**: Structures investment ideas into logical, high-conviction frameworks.
- **🚀 Multi-Catalyst Detection**: Automatically identifies internal and external events that could drive asset value.
- **🛡 Objective Risk Mapping**: Explicitly surfaces failure modes and risks to provide a balanced perspective.
- **✨ Universal Provider Support**: Compatible with OpenAI, Groq, Anthropic, and local Ollama instances.
- **📥 Dual-format Extraction**: Download your strategic reports in professional `.json` and `.txt` formats.
- **🎨 Elite Dashboard**: A premium Navy/Gold themed UI designed for venture capitalists, retail investors, and analysts.

---

## 🛠 Tech Stack

- **Backend**: Python 3.9+
- **AI Engine**: GPT-4o-mini (Default) / Multi-Provider Support
- **UI Framework**: Streamlit (Premium Financial Slate Aesthetic)
- **Environment**: Dotenv for secure API key management

---

## 🚀 Getting Started

### 1. Installation
```bash
pip install -r requirements.txt
```

### 2. Environment Setup
Add your API key to a `.env` file:
```bash
OPENAI_API_KEY='your-api-key-here'
```

### 3. Usage

#### **CapitalMind Dashboard (Recommended)**
High-conviction synthesis with a full graphical interface.
```bash
streamlit run app.py
```

#### **CLI Mode**
Fast batch processing of investment contexts.
```bash
python agent.py
```
