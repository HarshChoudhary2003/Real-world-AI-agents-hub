# 🧠 Skill Gap Analyzer Agent

The **Skill Gap Analyzer** is a strategic AI node in the **CareerOS** suite, designed to identify the exact technical and experience gaps between a candidate's resume and a target job role.

## 🚀 Core Intelligence: Strategic Mapping
This agent performs a deep-dive comparison and generates:
1.  **Match Score**: A quantitative percentage of overall job-candidate alignment.
2.  **Missing Fundamentals**: Identifies critical technical nodes (languages, tools, frameworks) that are missing from the resume.
3.  **Weak Areas**: Highlights areas where the resume shows some experience but lacks sufficient depth or clear impact evidence.
4.  **Actionable Learning Plan**: Provides specific, project-based strategies to bridge every detected gap.

## 🛠️ Features
- **Keyword Extraction**: Advanced parsing of Job Descriptions to extract required skills.
- **Project-Based Learning Paths**: Suggests concrete actions for every missing skill.
- **Multi-Model Intelligence**: Switch between OpenAI, Gemini, or Groq for varying analytical depth.
- **Glassmorphic UI**: High-density dashboard with visual indicators for match status.
