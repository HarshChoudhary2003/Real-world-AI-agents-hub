# 📊 Daily Reflection & Insight Agent (ProductivityOS)

The Daily Reflection Agent is a high-performance cognitive deconstruction architect. It closes the "Self-Improving AI Loop" by forensicly analyzing daily execution logs to extract tactical insights, emotional patterns, and surgical improvements for the siguiente performance node.

## 🧠 Core Intelligence
- **Behavioral Pattern Deconstruction**: Identifies the gap between planned missions and actual execution results.
- **Mood & Burnout Detection**: Utilizes linguistic audit technology to detect stress levels, burnout risks, and flow-state frequency.
- **Tactical Insight Synthesis**: Provides 3-5 high-impact coaching nodes based on the day's successes and failures.
- **Future-State Architecting**: Generates tomorrow's strategic focus nodes to ensure high-density execution.
- **Performance Streak Tracking**: Maintains a persistent habit-streak log to drive psychological consistency.

## 🚀 Key Features
- **🔥 Premium Glassmorphic UI**: High-density dark-themed dashboard with tactical-grid and performance visualizations.
- **📈 Historical Trend Charts**: Cross-session productivity score visualization for long-term growth monitoring.
- **🎭 Psychological State Audit**: Visual indicators for mood status and burnout risk warnings.
- **🧠 Mentor-Style Coaching**: Provides a "Coaching Pith" (high-impact advice) tailored to the day's performance.
- **⬇️ Intelligence Brief Export**: One-click download as a standardized Markdown node for personal journaling.

## 🛠 Setup & Launch
1. Clone the repository and navigate to this folder.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   streamlit run app.py
   ```
