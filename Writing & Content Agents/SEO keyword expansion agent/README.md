# 🔎 AI SEO Architect Pro

An enterprise-grade Semantic SEO Expansion agent that generates highly credible, data-rich keyword clusters suitable for Fortune 500 content strategy from a single seed keyword.

## 🌟 Features
* **Predictive Value Metrics**: The AI intuitively estimates Competition, Volume, Search Intent, and average CPC.
* **Cluster Architecture**: Automatically categorizes keywords into Primary, Semantic/LSI, Long-Tail, and PAAs (People Also Ask / Questions).
* **Multi-Provider Hub**: Works flawlessly with Anthropic Claude, OpenAI, and Google Gemini via safe initialization.
* **Enterprise Exports**: 1-Click universal CSV export that merges all clusters into a single unified Dataframe that can immediately be imported into Excel or Google Sheets.

## 🚀 Getting Started

### Prerequisites
Ensure you have access to your chosen AI vendor key.

### Installation
Install the required dashboard dependencies:
```bash
pip install openai anthropic google-generativeai streamlit pandas
```

### Running the App
Fire up the executive data dashboard:
```bash
streamlit run app.py
```

Or execute headless data runs:
```bash
python agent.py
```
