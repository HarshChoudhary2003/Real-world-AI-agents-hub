## ⚡ Overview

**SkillForge AI (Training Content Generator Agent)** allows HR Teams to feed in raw requirements (e.g., Target Audience, Macro Topic) and have an LLM output highly specific instructional modules. This eliminates the "Blank Page" problem for corporate enablement and instructional design.

## 🚀 Key Features

*   **🛡️ Dynamic Lexicon Routing:** Training generated for "New Sales Interns" will look fundamentally different than that generated for "Senior Database Architects."
*   **📊 Objective Extraction:** Forces abstract training requests into rigid, testable "Core Learning Objectives".
*   **💠 Executive Enablement UI:** A "Deep Violet & Slate" interface tuned explicitly to mimic premium Learning Management Systems (LMS).
*   **🧠 Total Orchestration:** Works over Litellm with any model you need—from `Claude-3.5-Sonnet` for writing elegance to `Llama 3` for entirely localized corporate secrets parsing.

## ⚙️ Quick Start

### 1. Execute CLI Pipeline
Inject the training brief into `training_input.txt` and execute:
```bash
cd "HR , Legal & Compliance Agents/Training content generator agent"
python agent.py
```

### 2. Enter Visual Interface
```bash
streamlit run app.py
```
*(Available locally on default port 8501-8518).*
