# 🔥 Habit Builder & Tracker Agent (ProductivityOS)

The Habit Builder Agent is a high-precision behavioral change architect designed to deconstruct macro-habits into atomic, high-consistency execution nodes. It bridges the gap between "Desired Action" and "Permanent Consistency" attraverso habit-stacking and dopaminergic gamification.

## 🧠 Core Intelligence
- **Behavioral Shift Architecture**: Deconstructs desired habits into zero-friction "Atomic Entry Steps."
- **Tactical Habit Stacking Mapping**: Surgically integrates new habits into existing daily routine nodes to ensure initiation.
- **Surgical Streak Persistence**: Utilizes loss-aversion and visual cues to architect high-fidelity streaks.
- **Dopaminergic Gamification Hub**: Points-based reward system (XP) designed to trigger neural dopamine release upon session success.
- **Motivation Nodes Swarm**: Provides real-time environmental and linguistic triggers for focus-drift recovery.

## 🚀 Key Features
- **🔥 Premium Glassmorphic UI**: High-density dark-themed dashboard with tactical-grid and gamified indicators.
- **📈 Persistence-Based XP Engine**: Persistent global XP tracking to drive long-term engagement and consistency.
- **📆 Phased Scaling Sequence**: Actionable, color-coded sequences for every timeline phase of the habit-transformation process.
- **🛡️ Streak Resilience Protcol**: AI-generated protocols for maintaining focus and recovering from streak-breaks.
- **🧬 Model Agnostic (LiteLLM)**: Fully compatible with GPT-4, Gemini Pro, Claude 3.5, and Llama 3 models.

## 🛠 Setup & Launch
1. Clone the repository and navigate to this folder.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   streamlit run app.py
   ```
