## ⚡ Overview

**ThinkForge AI** does not answer questions directly. Instead, it forces language models into algorithmic pathways—deconstructing complex logical riddles, physics equations, or scale problems into explicit steps, assumptions, calculations, and bounded constraints to reach a verified final deduction. 

## 🚀 Key Features

*   **🔍 Assumption Validation:** Binds the AI to explicitly state the boundaries of its world logic before attempting math.
*   **🔀 Step-by-Step Traversal:** Outputs distinct execution layers so operators can audit exactly where logic chains fail.
*   **🖥️ ThinkForge Web Terminal:** A violet-themed UI explicitly geared for entering deep mathematical string problems.
*   **🧩 Agnostic Core:** Leverage heavy-duty thinkers like `Claude 3.5 Sonnet` or `GPT-4o` instantly using unified parameters.

## ⚙️ Quick Start

### The Terminal Node
Drop your complex paradoxes into `problem.txt` and launch:
```bash
cd "Multi-step reasoning agent"
python agent.py
```

### Interactive Sandbox
Boot up the Streamlit interface locally:
```bash
streamlit run app.py
```
*Port defaults to 8501.*
