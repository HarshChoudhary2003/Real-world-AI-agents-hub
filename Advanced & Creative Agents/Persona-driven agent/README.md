**Identity.AI** is a high-level cognitive orchestration suite designed to project multi-agent intelligence through specialized, authoritative personas. 

### 🚀 Key Features
- **Persona Forge**: Real-time role-switching through predefined expert identities (Senior PM, Data Scientist, UX Architect).
- **Tone & Value Matrix**: Rigid adherence to role-specific constraints and professional business-focused language.
- **Zero Speculation Logic**: Strictly blocks non-authoritative output to maintain persona integrity.
- **Neural Matrix Studio**: A cyan-accented glassmorphism workspace for role-based mission projection.

### 🛠️ Tech Stack
- **AI**: GPT-4o-mini (Zero-deviation persona adhering).
- **Core**: Python 3.11+, Streamlit, `json`.

### ⚡ Quick Start
**To run the Identity Studio:**
```bash
streamlit run app.py
```
