## ⚡ Overview

**CleanCode AI** deterministically reads spaghetti backend or monolithic structural code blocks, processes them through heavy style guidelines, and completely refactors the architecture. Crucially: *It explicitly retains the exact same programmatic behavior*, optimizing solely for computational readability and standard software conventions.

## 🚀 Key Features

*   **🛡️ Execution Preservation:** Rules are hard-coded to ignore functional enhancements that break tests.
*   **📑 Reporting Pipelines:** Generates diff summaries detailing *why* variables or iterations were shifted, maintaining total transparency.
*   **💠 Elite "CodeForge" Dashboard:** Employs dual-pane text modules allowing users to review raw unoptimized snippets against polished `refactored_code.py`.
*   **🧠 Total Model Control:** Compatible with Llama 3 local endpoints, Gemini, OpenAI, Groq arrays, and Claude models.

## ⚙️ Quick Start

### Python Runtime
Point the script at your legacy modules via terminal payload:
```bash
cd "Code refactoring agent"
python agent.py
```

### Streamlit Launch
Execute the visual web dashboard:
```bash
streamlit run app.py
```
