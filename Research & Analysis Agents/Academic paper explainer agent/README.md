# 🎓 ScholarAI: Academic Paper Explainer Agent

**ScholarAI** is a high-performance AI agent designed to bridge the gap between complex academic research and accessible human understanding. Whether you're a curious student, a busy professional, or a researcher looking for a quick recap, ScholarAI distills dense scientific literature into structured, actionable insights in seconds.

---

## 🌟 Premium Features

- **🧠 Intelligent Synthesis**: Advanced NLP logic to extract core research questions and methodologies.
- **🏗 Structured Insights**: Generates structured reports covering Methodology, Key Findings, Implications, and Limitations.
- **✨ Premium UI/UX**: A dark-themed, glassmorphism-inspired interface built with Streamlit for a state-of-the-art experience.
- **📥 One-Click Export**: Download analysis in professional `.json` and `.txt` formats for integration into your workflow.
- **🛡 Grounded Analysis**: Built-in rules to avoid speculation and preserve the original nuance of the paper.

---

## 🛠 Tech Stack

- **Backend**: Python 3.9+
- **AI Engine**: OpenAI GPT-4o-mini (Optimized for speed/accuracy)
- **UI Framework**: Streamlit (Premium Custom CSS)
- **Data Management**: JSON/TXT structured exports

## 🚀 Getting Started

### 1. Installation
Clone the repository and install the high-performance dependencies:
```bash
pip install -r requirements.txt
```

### 2. Environment Setup
Create a `.env` file or set your system environment variable:
```bash
export OPENAI_API_KEY='your-api-key-here'
```

### 3. Launching the Agent

#### **Option A: Power Streamlit UI (Recommended)**
Experience the full graphical interface with live previews and export tools.
```bash
streamlit run app.py
```

#### **Option B: Batch CLI Mode**
Process papers directly via terminal for rapid-fire analysis.
```bash
python agent.py
```
