# 💰 Salary Negotiation Agent

The **Salary Negotiation Agent** is a high-leverage compensation architect in the **CareerOS** suite, designed to help candidates maximize their total reward packages across full-time and freelance roles.

## 🚀 Core Intelligence: The Leverage Engine
This agent moves the "CareerOS" mission from the application phase to the finalizing phase:
1.  **Market Benchmark Estimation**: Calculates the true market value of a role based on title, location, and experience nodes.
2.  **Negotiation Scripting**: Generates word-for-word email and verbal scripts tailored to the candidate's specific situation.
3.  **Style Selection**: Choose between **Polite**, **Assertive**, or **Confident** negotiation personas to match the seniority and company culture.
4.  **Offer Comparison Tool**: A built-in utility to compare the total value of multiple competing offers.

## 🛠️ Features
- **Total Rewards Strategy**: Negotiate beyond base salary to include bonuses, equity, and remote flexibility.
- **Pushback Logic**: Tactical advice on how to respond when a recruiter says "this is our final offer."
- **Multi-Model Intelligence**: Powered by LiteLLM for universal provider accessibility (OpenAI, Gemini, Groq, Anthropic).
- **Glassmorphic UI**: High-density dashboard with leverage metrics and script containers.
