## 🧩 Architectural Hierarchy

These agents are designed to work in a **Closed-Loop Ecosystem**:

1.  **Sensor Layer**: [📡 Sentinel.AI](./Long-running%20monitoring%20agent) detects system state.
2.  **Ethics Layer**: [⚖️ Ethics.AI](./Ethics%20&%20guardrails%20agent) evaluates the proposed maneuver.
3.  **Decision Layer**: [🧠 Decision AI](./Autonomous-decision-agent) computes the tactical resolution.
4.  **Governance Layer**: [⚖️ Governance.AI](./Human-in-the-loop-approval-agent) requests human authorization if risk is high.
5.  **Execution Layer**: [💠 Orchestra.AI](./Multi-agent%20collaboration%20system) coordinates the specialized agents.
6.  **Heal Layer**: [🩹 HealDesk.AI](./Error%20recovery%20agent) recovers if an error occurs.
7.  **Memory Layer**: [🧠 Neural.AI](./Memory-enabled%20agent) persists the outcome for future runs.
8.  **Optimization Layer**: [💠 Evolver.AI](./Self-improving%20agent) improves the logic based on the session feedback.

## 🚀 Deployment Instructions

Each agent in this category supports both **CLI** and **Streamlit Dashboard** modes.

**To run an agent's dashboard:**
```bash
cd "Advanced & Creative Agents/[Agent Folder]"
streamlit run app.py
```

**To run an agent's CLI engine:**
```bash
python agent.py
```
