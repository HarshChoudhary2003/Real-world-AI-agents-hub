# ✉️ Cover Letter Generator AI Agent

A professional AI agent designed to bridge the gap between a candidate's resume and a specific job description by crafting high-conversion, personalized cover letters.

## 🧠 Features
- **Persona-Driven Writing**: Supports multiple tones (Professional, Bold, Creative, etc.).
- **Strategic Focus**: Tailor the narrative towards Technical Expertise, Leadership, or Culture Fit.
- **Multi-Variant Generation**: Generate up to 3 unique variations to choose the best fit.
- **Glassmorphic UI**: Premium, high-density dashboard for a seamless user experience.

## 🛠️ Tech Stack
- **Core**: OpenAI GPT-4o-mini
- **Frontend**: Streamlit (Premium CSS)
- **Logic**: Python

## 🚀 Quick Start
1. Ensure you have an `OPENAI_API_KEY` in your environment.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the agent:
   ```bash
   streamlit run app.py
   ```
