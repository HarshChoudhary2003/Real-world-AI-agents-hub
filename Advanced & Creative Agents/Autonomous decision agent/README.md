**Decision AI** is a deterministic decision-making engine that processes complex contextual inputs into structured, executable maneuvers. 

### 🚀 Key Features
- **Deterministic Rationale**: Ensures every decision is backed by explainable logic and trade-off analysis.
- **Risk Indexing**: Automatically classifies decisions as Low, Medium, or High risk.
- **Neural Studio**: A high-contrast "Alert Red" dashboard for mission intervention.
- **Structured JSON Logic**: Produces machine-readable outputs for automated systems.

### 🛠️ Tech Stack
- **AI**: GPT-4o-mini (Zero-temperature for maximum consistency).
- **Core**: Python 3.11+, Streamlit, `openai`, `json`.

### ⚡ Quick Start
**To run the Decision Studio:**
```bash
streamlit run app.py
```
