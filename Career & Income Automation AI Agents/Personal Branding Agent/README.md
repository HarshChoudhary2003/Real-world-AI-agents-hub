# 🌐 Personal Branding Agent

The **Personal Branding Agent** is a strategic identity architect in the **CareerOS** suite, designed to transform a candidate's resume and achievements into a high-impact digital presence.

## 🚀 Core Intelligence: The Narrative Engine
This agent moves the "CareerOS" mission from applying to showcasing:
1.  **LinkedIn Accelerator**: Generates value-driven, high-hook LinkedIn posts to build authority.
2.  **GitHub Profile README**: Crafts a professional Markdown summary to showcase technical entropy.
3.  **Portfolio Bio**: Drafts high-fidelity "About Me" sections for personal websites.
4.  **Branding Strategy**: Delivers a step-by-step roadmap to building professional authority and visibility.

## 🛠️ Features
- **Narrative Alignment**: Syncs your online identity with your target authority roles.
- **Copy-to-Clipboard**: Direct results ready for LinkedIn and GitHub profile updates.
- **Multi-Model Intelligence**: Powered by LiteLLM for universal primary provider support.
- **Glassmorphic UI**: High-density dashboard with visual authority indicators.
