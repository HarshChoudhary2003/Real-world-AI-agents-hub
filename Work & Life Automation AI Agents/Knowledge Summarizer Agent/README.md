# 📚 Knowledge Summarizer Agent (ProductivityOS)

The Knowledge Summarizer Agent is a high-fidelity deconstruction engine designed to transform high-density content (articles, research, or notes) into surgical, actionable tactical intelligence.

## 🧠 Core Intelligence
- **Multi-Dimensional Knowledge Synthesis**: Deconstructs narrative content into its "First Principles" and neural summaries.
- **Adaptive Compression Modes**: Switch between "Short & Punchy" (high-density) and "Detailed Synthesis" (comprehensive mapping).
- **Execution-Ready Takeaways**: Extracts tactical application steps that can be immediately integrated into the ProductivityOS workflow.
- **Learning Acceleration Protocols**: Provides deeper insights and related mental models to connect new knowledge with existing systems.
- **Micro-Node Deconstruction**: Breaks large content clusters into surgical knowledge-points.

## 🚀 Key Features
- **🔥 Premium Glassmorphic UI**: High-density dark-themed dashboard with visual knowledge-grid layout.
- **📂 Multi-Format Ingestion**: Supports direct text pasting or file uploads (.txt, .md).
- **🎓 Learning Accelerator Swarm**: AI-generated suggestions for mastering topics and identifying context depth.
- **⬇️ Knowledge Brief Export**: One-click download as a clean, standardized text brief for distribution.
- **🧬 Model Agnostic (LiteLLM)**: Fully compatible with GPT-4, Gemini Pro, Claude 3.5, and Llama 3 models.

## 🛠 Setup & Launch
1. Clone the repository and navigate to this folder.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   streamlit run app.py
   ```
