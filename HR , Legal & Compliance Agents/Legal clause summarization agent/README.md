## ⚡ Overview

**BriefForge AI (Legal Clause Summarization Agent)** acts as an immediate syntax parser for heavy contract clauses. It guarantees the core legal mechanism of the text is retained but translated into executive language for non-lawyers.

## 🚀 Key Features

*   **🛡️ Obligation Extraction:** Accurately isolates "who must do what" and specifically drops action-based rules into distinct UI cards.
*   **📊 Risk Structuring:** Instantly extracts the precise downside or conditional liability within "Limitation" clauses.
*   **💠 Executive Translation UI:** A "Deep Slate & Gold" UI heavily optimized to feel like a high-end counsel terminal.
*   **🧠 Total Orchestration:** Works over Litellm with any model you need—from `Claude-3.5-Sonnet` for writing elegance to `Llama 3` for entirely private, secure contract parsing.

## ⚙️ Quick Start

### 1. Execute CLI Pipeline
Inject the contract payload into `clause.txt` and execute:
```bash
cd "HR , Legal & Compliance Agents/Legal clause summarization agent"
python agent.py
```

### 2. Enter Visual Interface
```bash
streamlit run app.py
```
*(Available locally on default port 8501-8516).*
