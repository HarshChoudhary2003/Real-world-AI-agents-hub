# 🎼 Orchestra-Core AI (Workflow Orchestration)

Architect high-performance execution paths for complex multi-step systems with neural dependency resolution and surgical recovery blueprints.

## 🚀 Overview
**Orchestra-Core AI v2.0** is an enterprise-grade Systems Orchestration OS. It goes beyond simple task sequencing, using multi-model intelligence to resolve **topological dependencies**, identify **circular loops**, and architect **Surgical Recovery Blueprints** for every potential failure point in your pipeline.

## ⚡ Key Features
- **Neural Dependency Resolution**: Automatically calculates the optimal Path-to-Resolution for complex, interleaved workflows.
- **Parallel Grouping Diagnostics**: Identifies steps that can be run concurrently to maximize system throughput and minimize latency.
- **Surgical Recovery Blueprints**: Anticipates specific incidents (Auth failures, timeouts, DB locks) and provides remediation plans.
- **Bottleneck Forensics**: Pinpoints the "Path Anchors" (Critical nodes) that block overall workflow efficiency.
- **Multi-Provider Architecture**: Native selection of your intelligence brain (OpenAI, Gemini, Claude, DeepSeek, or Groq).

## 🛠️ Tech Stack
- **Frontend**: Streamlit (Premium Creative UI with Priority Node Cards)
- **Intelligence**: LiteLLM (Multi-model support)
- **Data Protocols**: JSON for architectural object export, TXT for execution summaries.

## 📂 Structure
- `agent.py`: Core orchestration engine and multi-provider CLI wrapper.
- `app.py`: Premium animated Streamlit dashboard with component analytics.
- `workflow.json`: Sample multi-step workflow for testing.
- `requirements.txt`: Project dependencies.

## 🚀 Quick Launch

### 1. CLI Usage
```bash
python agent.py
```

### 2. Dashboard Usage
```bash
streamlit run app.py
```

## 📊 Strategic Output
The agent outputs a structured JSON analysis including:
- **Orchestra Assessment**: An efficiency score, a status verdict, and a technical diagnostic summary.
- **Execution Blueprint**: Prioritized steps with parallelization group assignments.
- **Recovery Blueprints**: Surgical remediation plans for anticipated failure scenarios.
