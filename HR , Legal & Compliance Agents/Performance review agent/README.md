## ⚡ Overview

**ReviewForge AI (Performance Review Agent)** allows managers to input unstructured, potentially overly-direct, or fragmented bullet points about an employee's quarterly progress. The neural core processes this raw logic and outputs a highly professional, constructive, and action-oriented review spanning `Strengths`, `Areas for Improvement`, and `Action Plans`.

## 🚀 Key Features

*   **🛡️ Liability Filtering:** Prevents emotionally charged or inappropriately biased feedback by translating harsh reviews into objective 'Growth Areas'.
*   **📊 Structured Action Plans:** Does not just list weaknesses, it logically infers and generates required `development_recommendations` for the next quarter.
*   **💠 Executive Dark Mode UI:** A professional Deep Navy/Azure styled web interface prioritizing fast tab-switching to compare performance factors.
*   **🧠 Total Orchestration:** Works over Litellm with any model you need—from `Claude-3.5-Sonnet` for writing elegance to `Llama 3` for local privacy.

## ⚙️ Quick Start

### 1. Execute CLI Pipeline
Inject management points into `review_input.txt` and execute:
```bash
cd "HR , Legal & Compliance Agents/Performance review agent"
python agent.py
```

### 2. Enter Visual Interface
```bash
streamlit run app.py
```
*(Available locally on default port 8501-8512).*
