# 🎯 Goal Breakdown & Strategy Agent (ProductivityOS)

The Goal Breakdown Agent is a high-density strategic consultant designed to deconstruct macro-goals into surgical, actionable micro-sequences. It eliminates ambiguity and provides a clear architectural roadmap for rapid execution.

## 🧠 Core Intelligence
- **Strategic Milestone Engineering**: Identifies the high-impact signal-nodes that indicate real progress.
- **Phased Execution Architecture**: Breaks down timelines into logical focus-pockets (e.g., Sprint Alpha, Accelerate).
- **Temporal Resilience Protocol**: Incorporates failure-recovery nodes and adaptive strategies for when momentum drifts.
- **Visual Roadmap Synthesis**: Generates a high-precision execution sequence with a visual progress architecture.
- **Tactical First-Step Extraction**: Pinpoints the literal first actions needed to initiate mission momentum.

## 🚀 Key Features
- **🔥 Premium Glassmorphic UI**: High-density dark-themed dashboard with tactical-grid layout.
- **📈 Visual Progress Tracker**: Integrated line-chart tracking of roadmap milestones.
- **📋 Phase-Specific Task Cards**: Color-coded, prioritized sequence of execution nodes for every timeline phase.
- **🛡️ Resilience Analysis**: AI-generated protocols for maintaining focus and recovering from execution stalls.
- **🧬 Model Agnostic (LiteLLM)**: Fully compatible with GPT-4, Gemini Pro, Claude 3.5, and Llama 3 models.

## 🛠 Setup & Launch
1. Clone the repository and navigate to this folder.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   streamlit run app.py
   ```
