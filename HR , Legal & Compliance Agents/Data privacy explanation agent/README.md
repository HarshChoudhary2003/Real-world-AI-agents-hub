## ⚡ Overview

**PrivacyForge AI (Data Privacy Explanation Agent)** gives product teams and engineering leads an immediate translation layer between the features they want to build and the restrictive international data privacy laws governing those features. 

## 🚀 Key Features

*   **🛡️ Intent Parsing:** Instead of just sending you a multi-page GDPR manual, PrivacyForge acts on specific scenarios (e.g., *"We want to collect EU emails"*).
*   **📊 Matrix Distillation:** Breaks legal consequences down into `Corporate Obligations`, `Individual End-User Rights`, and `Practical Implementation Implications`.
*   **💠 Executive Teal UI:** Built atop a deep Obsidian & Cyan dark mode perfectly suited for continuous operational risk monitoring and system auditing.
*   **🧠 Total Orchestration:** Works over Litellm with any model you need—from `Claude-3.5-Sonnet` for writing elegance to `Llama 3` for absolutely private compliance extraction.

## ⚙️ Quick Start

### 1. Execute CLI Pipeline
Inject the operational requirement into `privacy_input.txt` and execute:
```bash
cd "HR , Legal & Compliance Agents/Data privacy explanation agent"
python agent.py
```

### 2. Enter Visual Interface
```bash
streamlit run app.py
```
*(Available locally on default port 8501-8517).*
