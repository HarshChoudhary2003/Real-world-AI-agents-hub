# 🚑 Error-Forensics AI (Error Classification)

Perform deep linguistic and logical forensics on system incidents to identify root causes, impact severity, and architect surgical code-level remediations.

## 🚀 Overview
**Error-Forensics AI v2.0** is an enterprise-grade Site Reliability Engineering (SRE) OS. It goes beyond simple categorizing, using multi-model intelligence to resolve **incident patterns**, calculate **impact diagnostics**, and architect **Surgical Remediation Blueprints** for production-grade systems.

## ⚡ Key Features
- **Multi-Vector Incident Auditing**: Automatically classifies errors by Category (DB, Network, Logic), Severity (Critical to Low), and Impact (User vs. System).
- **Forensic Root Cause Analysis**: Identifies the "Mechanical Why" of an incident (e.g., protocol timeouts, deadlocks, resource leaking).
- **Surgical Remediation Blueprints**: Provides specific, actionable "Immediate Fixes" and long-term "Structural Architecture" enhancements.
- **SRE Standards & Monitoring**: Suggests specific logging and metrics strategies (e.g., Datadog, Sentry, Prometheus) to detect future occurrences.
- **Multi-Provider Architecture**: Native selection of your intelligence brain (OpenAI, Gemini, Claude, DeepSeek, or Groq).

## 🛠️ Tech Stack
- **Frontend**: Streamlit (Premium Creative UI with Pulsing Severity Indicators)
- **Intelligence**: LiteLLM (Multi-model support)
- **Data Protocols**: JSON for diagnostic object export, TXT for incident summaries.

## 📂 Structure
- `agent.py`: Core forensic engine and multi-provider CLI wrapper.
- `app.py`: Premium animated Streamlit dashboard with component forensics.
- `error.txt`: Sample system error log for testing.
- `requirements.txt`: Project dependencies.

## 🚀 Quick Launch

### 1. CLI Usage
```bash
python agent.py
```

### 2. Dashboard Usage
```bash
streamlit run app.py
```

## 📊 Strategic Output
The agent outputs a structured JSON analysis including:
- **Audit Report**: A severity level, a primary category, and a strategic summary.
- **Forensic Breakdown**: Technical explanation of root causes and impact assessments.
- **Remediation Blueprints**: Actionable fix guides for developers.
