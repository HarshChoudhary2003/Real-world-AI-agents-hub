# 📊 Alert-Insight AI (Monitoring Alert Explanation)

Translate cryptic system alerts from Datadog, Prometheus, Sentry, or CloudWatch into plain-English root causes and remediation blueprints with neural context resolution.

## 🚀 Overview
**Alert-Insight AI v2.0 (SaaS Edition)** is a premium Observability Intelligence OS. It goes beyond simple alert relaying, using multi-model intelligence to resolve **incident criticality**, identify **systemic causes**, and architect **Remediation Roadmaps** for infrastructure and application service responders.

## ⚡ Key Features
- **Neural Alert Interpretation**: Automatically translates technical threshold breaches into human-readable executive briefings.
- **Surgical Cause Isolation**: Specifically pinpoints the mechanical reason (Network handshake, DB deadlock, IO wait) for a monitor trigger.
- **Remediation Roadmap**: Provides prioritized "Immediate Action" steps and long-term "Structural Optimizations."
- **Monitor Tuning Intelligence**: Recommends adjustments to alert thresholds and tags to minimize noise and reach observability excellence.
- **Multi-Provider Architecture**: Native selection of your intelligence brain (OpenAI, Gemini, Claude, DeepSeek, or Groq).

## 🛠️ Tech Stack
- **Frontend**: Streamlit (SaaS Elite UI with Glassmorphism, Top/Sidebar Nav, and Plotly Analytics)
- **Intelligence**: LiteLLM (Multi-model support)
- **Data Protocols**: JSON for incident export, TXT for briefing reports.

## 📂 Structure
- `agent.py`: Core interpretation engine and multi-provider CLI wrapper.
- `app.py`: Premium SaaS-style Streamlit dashboard with component forensics.
- `alert.txt`: Sample monitoring alert for testing.
- `requirements.txt`: Project dependencies.

## 🚀 Quick Launch

### 1. CLI Usage
```bash
python agent.py
```

### 2. Dashboard Usage
```bash
streamlit run app.py
```

## 📊 Strategic Output
The agent outputs a structured JSON analysis including:
- **Alert Briefing**: Criticality, title, and plain-English summary.
- **Diagnostic Deep-Dive**: Core cause, impacted tags/regions, and technical details.
- **Remediation Roadmap**: Step-by-step fix guides for responders.
