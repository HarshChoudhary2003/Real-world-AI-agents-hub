## ⚡ Overview

**ArchForge AI** enables lead developers to extract logical structures—such as microservice relationships, database intent, and communication bottlenecks—from simple unstructured notes or meeting transcripts, formatting them into professional architectural system overview blocks immediately.

## 🚀 Key Features

*   **⚖️ Trade-off Intelligence:** Identifies potential scale or bottleneck problems inherently introduced by the extracted structures.
*   **🔄 Data Flow Modeling:** Summarizes high-level system routing into comprehensive pathways.
*   **💎 ArchForge Dashboard:** A striking Purple/Deep Slate UI designed entirely for rapid system decomposition.
*   **🧠 Ecosystem Agnosticism:** Easily integrate open-weights like DeepSeek or LLama through local Ollama servers—all handled by LiteLLM via Progressive Selection Inputs.

## ⚙️ Quick Start

### Start the Analyzer

Drop abstract logic or whiteboard transcripts into `architecture.txt` and execute:
```bash
python agent.py
```

### Launch the UI Console
```bash
streamlit run app.py
```
