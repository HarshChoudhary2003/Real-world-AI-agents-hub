# 📧 Email Assistant Agent (ProductivityOS)

The Email Assistant is a high-impact tactical communication architect. It transforms messy contexts, unstructured situations, or raw reply-threads into polished, high-conversion email drafts tailored precisely to your desired tone and objective.

## 🧠 Core Intelligence
- **Tactical Context Mapping**: Understands complex situations and threads to generate coherent, non-repetitive responses.
- **Subject Line Engineering**: Crafts high-open-rate subject lines designed for professional elite communication.
- **Smart Reply Swarm (3 Variants)**: Automatically generates three different versions for every request (Direct & Concise, Detailed & Explanatory, Persuasive & Impactful).
- **Linguistic Grammar Check**: Provides real-time polishing suggestions for flow, tone, and grammar.
- **Tone Chronomics**: Adjusts the communication style based on a 4-tier tone scale (Strictly Formal to Confident/Direct).

## 🚀 Key Features
- **🔥 Premium Glassmorphic UI**: High-density dark-themed dashboard with tactical-grid layout.
- **📋 Multi-Variant Tabs**: Switch instantly between different draft versions and choose the best one.
- **✨ Grammar Policing**: Identifies passive voice and redundant phrasing for professional polishing.
- **⬇️ Tactical Draft Export**: One-click download as a clean, standardized text brief for distribution.
- **🧬 Model Agnostic (LiteLLM)**: Fully compatible with GPT-4, Gemini Pro, Claude 3.5, and Llama 3 models via Groq.

## 🛠 Setup & Launch
1. Clone the repository and navigate to this folder.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   streamlit run app.py
   ```
