# 🎬 StreamBrief: YouTube Video Summary Agent

[![YouTube](https://img.shields.io/badge/Platform-YouTube-red?style=for-the-badge&logo=youtube)](https://youtube.com)
[![Expert AI](https://img.shields.io/badge/AI-Summary_Expert-blueviolet?style=for-the-badge&logo=openai)](https://openai.com/)
[![Market-Ready](https://img.shields.io/badge/Status-Market_Ready-success?style=for-the-badge)](https://github.com/HarshChoudhary2003/Real-world-AI-agents-hub)

**StreamBrief** is a high-performance AI agent designed to extract structured intelligence from YouTube transcripts. It filters out filler, identifies core arguments, and provides a clear directional takeaway, saving hours of manual viewing time.

---

## 🌟 Premium Features

- **🧠 Logical Synthesis**: Uses advanced NLP to distinguish between main arguments and supporting examples.
- **🏗 Structured Insights**: Generates structured reports covering Overview, Key Points, Examples, and Final Takeaways.
- **✨ Universal API Support**: Built-in support for OpenAI, Groq, OpenRouter, DeepSeek, and Local LLMs (Ollama).
- **📥 Dual Export**: One-click download in `.json` and `.txt` formats.
- **🛡 Grounded Summary**: Strict rules to prevent AI hallucination or personal interpretation.

---

## 🛠 Tech Stack

- **Backend**: Python 3.9+
- **AI Engine**: GPT-4o-mini (Optimized) / Custom Providers
- **UI Framework**: Streamlit (Premium Glassmorphism Design)

---

## 🚀 Getting Started

### 1. Installation
```bash
pip install -r requirements.txt
```

### 2. Environment Setup
Create a `.env` file or set your API key:
```bash
export OPENAI_API_KEY='your-api-key'
```

### 3. Usage

#### **Power Streamlit UI (Recommended)**
```bash
streamlit run app.py
```

#### **CLI Mode**
```bash
python agent.py
```
