**Neural.AI** is a memory-augmented intelligence swarm that persists context, facts, and interaction patterns across an infinite operational horizon. 

### 🚀 Key Features
- **Long-Term Memory Vault**: Persistent storage of user facts and historical interactions in `memory.json`.
- **Selective Context Retrieval**: Neural engine dynamically extracts relevant records to personalize current responses.
- **Adaptive Interaction Studio**: Conversation-style workspace focused on continuous learning.
- **Privacy-First Governance**: Strict rules for avoiding sensitive data storage while maintaining a 20-entry sliding window.

### 🛠️ Tech Stack
- **AI**: GPT-4o-mini (Strategic context injection).
- **Core**: Python 3.11+, Streamlit, `json`.

### ⚡ Quick Start
**To run the Neural Studio:**
```bash
streamlit run app.py
```
