# 📰 NewsFlow: News Aggregation Agent

**NewsFlow** is a high-performance AI agent designed to distill the chaotic global news cycle into structured, thematic intelligence. It helps professionals, researchers, and decision-makers stay updated on specific sectors without the noise of sensationalist media.

---

## 🌟 Premium Features

- **🌐 Multi-Thematic Synthesis**: Automatically groups news into logical buckets (e.g., Regulation, Innovation, Market Shifts).
- **🏗 Executive Digest**: Generates high-level summaries alongside deep-dive thematic developments.
- **✨ universal Provider Support**: Compatible with OpenAI, Groq, OpenRouter, DeepSeek, and local Ollama instances.
- **📥 Dual-format Export**: Download reports in professional `.json` and `.txt` formats.
- **🛡 Neutral Filtering**: Prompts are engineered to eliminate bias, filler, and clickbait.

---

## 🛠 Tech Stack

- **Backend**: Python 3.9+
- **AI Engine**: GPT-4o-mini (Default) / Any custom provider
- **UI Framework**: Streamlit (Premium Slate/Blue News Aesthetic)

---

## 🚀 Getting Started

### 1. Installation
```bash
pip install -r requirements.txt
```

### 2. Environment Setup
Set your API key in a `.env` file or environment variable:
```bash
export OPENAI_API_KEY='your-api-key'
```

### 3. Usage

#### **Power Streamlit UI (Recommended)**
Experience the full graphical interface with thematic cards and export tools.
```bash
streamlit run app.py
```

#### **CLI Mode**
```bash
python agent.py
```
