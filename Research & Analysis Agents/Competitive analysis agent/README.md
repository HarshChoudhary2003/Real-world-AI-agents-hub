# Competitive Analysis Agent 📊🛡️

An enterprise-grade strategic intelligence tool designed to objectively analyze market competitors, identify strategic gaps, and extract actionable opportunities.

## Features ✨

- **Multi-Model Intelligence**: Leverages GPT-4o, Claude 3.5, and Llama 3 for deep strategic synthesis.
- **Strategic Matrix**: Breakdown of competitor strengths, weaknesses, and positioning.
- **Enterprise Dashboard**: premium Streamlit UI with glassmorphism aesthetics.
- **Actionable Insights**: Focuses on identifying market gaps and untapped opportunities.

## Usage 🚀

1. **Dashboard**:
   ```bash
   streamlit run app.py
   ```
2. **CLI**:
   ```bash
   python agent.py --provider openai --model gpt-4o
   ```
