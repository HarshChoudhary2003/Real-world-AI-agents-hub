# ✍️ Product Copywriter Pro

An elite, high-converting Product Description architect that ingests generic technical features and perfectly maps them to tangible, real-world benefits for an exact target audience.

## 🌟 Features
* **Benefit-Driven Mapping**: An AI Chain-of-Thought pipeline that prevents hype/magic-pill claims and strictly relies on tangible outcome mappings.
* **Premium Dashboard**: A beautifully designed Slate and Deep Blue UI featuring custom CTA boxes and Benefit-Chip renderers.
* **Universal SDK Support**: Out-of-the-box support for the newest APIs from OpenAI, Google, and Anthropic.
* **Direct Drafting**: 1-click text exports generate exact copy-and-paste ready product drafts.

## 🚀 Getting Started

### Installation
Install the core NLP SDKs along with the Streamlit frontend.
```bash
pip install openai anthropic google-generativeai streamlit
```

### Running the App
Run the Sales Architecture Dashboard:
```bash
streamlit run app.py
```

Operate the silent CLI backend system:
```bash
python agent.py
```
