<br>

**Evolver.AI** is a feedback-driven self-correction engine that autonomously applies optimizations to its own logic based on historical performance outcomes. 

### 🚀 Key Features
- **Neural Feedback Mapping**: Ingests `feedback.json` to identify issues (verbosity, missing metrics, etc.).
- **Controlled Neural Evolution**: Applies incremental optimizations to the task output while respecting constraints.
- **Optimization Roadmap**: A visual checklist of exactly which self-corrections were applied.
- **Improved Analytics Hub**: KPI metrics showing improvement velocity and health index.

### 🛠️ Tech Stack
- **AI**: GPT-4o-mini (Self-correction reasoning).
- **Core**: Python 3.11+, Streamlit, `json`.

### ⚡ Quick Start
**To run the Optimization Studio:**
```bash
streamlit run app.py
```
