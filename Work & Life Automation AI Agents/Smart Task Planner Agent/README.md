# ⚡ Smart Task Planner Agent (ProductivityOS)

The Smart Task Planner is a high-performance daily strategy architect. It transforms vague goals into a surgical, time-blocked execution plan, optimized for the user's focus cycles and available energy.

## 🧠 Core Intelligence
- **Surgical Time Blocking**: Allocates specific slots based on biological peak focus nodes (90-min cycles).
- **Eisenhower Priority Hierarchy**: Automatically categorizes tasks based on High, Medium, or Low urgency.
- **Micro-Focus Suggestions**: Injects psychology-based focus tips (e.g., Pomodoro, Eat the Frog).
- **Streak & Performance Recovery**: Integrated streak tracker to build habit momentum.

## 🚀 Key Features
- **🔥 Glassmorphic UI**: Premium, dark-themed interface for modern high-performers.
- **⏱ Time Allocation Slider**: Define exactly how much deep work vs. shallow work time you have.
- **✅ Task Execution System**: Mark tasks as done and track progress across sessions.
- **📊 Daily Productivity Score**: AI-calculated Focus Score for each plan.
- **💡 Focus Tips & Daily Mantras**: Motivation and tactics for high-level execution.
- **🧬 Model Agnostic (LiteLLM)**: Support for GPT-4, Gemini Pro, Claude 3, and Llama 3 via Groq.

## 🛠 Setup & Launch
1. Clone the repository and navigate to this folder.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   streamlit run app.py
   ```
