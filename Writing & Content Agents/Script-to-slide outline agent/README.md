# 📽️ AI Presentation Architect Pro

An uncompromising AI system specifically trained to ingest chaotic, dense, long-form scripts, blogs, or speeches and synthesize them into perfectly paced, highly persuasive visual presentation slide outlines.

## 🌟 Features
* **Cognitive Load Formatting**: The AI natively avoids 'Slideuments' (overcrowded slides) by enforcing strict word-counts per bullet and strict idea-limits per slide.
* **Premium Dashboard UI**: Designed with a highly modern Teal & Cyan aesthetic, formatting the output natively into distinct, interactive visual "Slide" cards in the browser.
* **Multi-Provider Hub**: Supports flawless execution across OpenAI (`gpt-4o`, `gpt-4o-mini`), Google Gemini (`gemini-1.5-pro`, `gemini-1.5-flash`), and Anthropic Claude (`claude-3-5-sonnet`).
* **Universal Export**: 1-click download of the JSON payload or a highly structured Markdown text file.

## 🚀 Getting Started

### Prerequisites
Make sure you have your desired API keys:
- `OPENAI_API_KEY`
- `ANTHROPIC_API_KEY`
- `GEMINI_API_KEY`

### Installation
Ensure you install the core NLP libraries and Streamlit:
```bash
pip install openai anthropic google-generativeai streamlit
```

### Running the App
Run the interactive Streamlit Dashboard:
```bash
streamlit run app.py
```

Run the headless CLI Pipeline:
```bash
python agent.py
```
