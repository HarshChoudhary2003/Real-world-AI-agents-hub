<br>

**Sentinel.AI** is an autonomous long-running telemetry monitor that tracks system metrics and triggers alerts based on persistent threshold violations. 

### 🚀 Key Features
- **Rolling Telemetry Buffer**: Maintains a real-time 100-entry log of system health.
- **Threshold-Aware Alerting**: Triggers ONLY when an anomaly persists for X consecutive checks.
- **Live Performance Dashboard**: Visual charts for CPU, Memory, and Custom Metrics.
- **Autonomous Feedback Loop**: Corrects its own baseline over long durations.

### 🛠️ Tech Stack
- **AI**: GPT-4o-mini (Self-correction intelligence).
- **Core**: Python 3.11+, `psutil`, Streamlit, `pandas`.

### ⚡ Quick Start
**To run the Sentinel Dashboard:**
```bash
streamlit run app.py
```
