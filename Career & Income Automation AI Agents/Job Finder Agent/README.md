# 🔎 Job Finder Agent

The ultimate search engine and evaluator for your career job search, part of the **CareerOS** suite.

## 🧠 Core Features
- **Global Job Extraction**: Real-time job results from JSearch RapidAPI (LinkedIn, Indeed, Glassdoor via a single source).
- **AI Match Score**: Paste your resume to see an instant match percentage calculated by GPT-4o-mini.
- **Role & Location Precision**: Target any job title specifically, anywhere in the world.
- **Multi-Model Intelligence**: Choose between OpenAI Core or Google Gemini for your match scoring.
- **Glassmorphic UI**: High-density dashboard with direct apply links and company logo extraction.

## 🛠️ Usage
1. Provide a **RapidAPI (JSearch) Key** in the sidebar.
2. (Optional) Provide an **OpenAI API Key** for match scoring.
3. Paste your **Resume** text.
4. Input your target **Role** and **Location**.
5. Let the AI find and evaluate every opportunity for you.

## 🚀 Quick Launch
```bash
pip install -r requirements.txt
streamlit run app.py
```
