# 🎯 Focus Coach Agent (ProductivityOS)

The Focus Coach is a cognitive performance architect designed to bridge the gap between "Planning" and "Execution." It provides the environmental and psychological structure needed to achieve and maintain a flow state (Deep Work).

## 🧠 Core Intelligence
- **Flow State Orchestration**: Suggests ritualized triggers to initiate deep focus sessions based on task complexity.
- **Surgical Distraction Suppression**: Identifies specific environmental and internal blockers and provides tactical counters.
- **Micro-Node Deconstruction**: Breaks large missions into manageable 25-50 minute execution blocks.
- **Energy-Responsive Planning**: Adjusts workload intensity based on the user's current "Biological Energy Reservoir."

## 🚀 Key Features
- **🔥 Premium Glassmorphic UI**: High-fidelity dark mode dashboard with focus-centric layout.
- **⏱ Integrated Pomodoro Timer**: Live focus-block tracking directly within the interface.
- **📊 Concentration Depth Score**: AI-projected focus score for your current session strategy.
- **⚡ Deep Work Mode**: A UI toggle that signals environmental isolation and mission focus.
- **🛡️ Distraction Shielding**: Actionable advice for multi-tasking suppression and focus-drift recovery.

## 🛠 Setup & Launch
1. Clone the repository and navigate to this folder.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   streamlit run app.py
   ```
