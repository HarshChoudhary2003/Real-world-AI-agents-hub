**Governance.AI** provides a high-fidelity human authority gateway for authorized maneuvers in mission-critical autonomous workflows. 

### 🚀 Key Features
- **Neural Request Formulation**: Formats complex operational proposals into human-readable approval requests.
- **Risk-Aware UI**: Dynamic color-coded badges (High/Medium/Low) for rapid risk triage.
- **Decision Audit Vault**: An immutable log file (`approval_output.json`) records every human decision.
- **Enforcement Modal**: Strategic Authorize/Reject controls for high-stakes workflows.

### 🛠️ Tech Stack
- **AI**: GPT-4o-mini (Proposal synthesis).
- **Core**: Python 3.11+, Streamlit, `json`, `datetime`.

### ⚡ Quick Start
**To run the Approval Hub:**
```bash
streamlit run app.py
```
