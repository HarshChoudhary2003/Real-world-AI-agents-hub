# 💰 Freelance Proposal Agent

The **Freelance Proposal Agent** is a strategic income architect in the **CareerOS** suite, designed to transform complex gig descriptions into high-converting, winning proposals.

## 🚀 Core Intelligence: The Income Engine
This agent moves the "CareerOS" mission from learning to earning:
1.  **Platform-Specific Optimization**: Tailored pitches for Upwork, Fiverr, and Freelancer.
2.  **Conversion Psychology**: Injects trust signals, empathy, and urgency into every proposal.
3.  **Pricing Strategy**: Detailed advice on how to bid (low vs. high) and justifying the fee.
4.  **Tone Selector**: Switch between Expert, Friendly, or Aggressive tones to match the client's personality.

## 🛠️ Features
- **Client Psychology Analysis**: Deconstructs the client's "unspoken" needs based on the job description.
- **Key Selling Points**: Distills your resume into 3-4 bulletproof arguments for why you're the best fit.
- **Multi-Model Intelligence**: Powered by LiteLLM for universal provider accessibility.
- **Glassmorphic UI**: High-density dashboard with direct proposal downloads.
