# 💬 AI FAQ Generator Pro

An uncompromising, enterprise-grade AI agent designed to extract the core concerns, objections, and friction points of any given product, and synthesize them into highly credible, transparent Frequently Asked Questions (FAQs).

## 🌟 Features
* **Objection Handling Core**: Bypasses fluffy marketing jargon to deliver brutally honest, reassuring answers to real user concerns.
* **Premium Dashboard UI**: Designed with a highly professional Violet & Deep Purple aesthetic, complete with tactile, interactive FAQ hover cards.
* **Multi-Provider Hub**: Supports flawless execution across OpenAI (`gpt-4o`, `gpt-4o-mini`), Google Gemini (`gemini-1.5-pro`, `gemini-1.5-flash`), and Anthropic Claude (`claude-3-5-sonnet`).
* **Universal Export**: 1-click download of the JSON payload or a highly structured Markdown text file.

## 🚀 Getting Started

### Prerequisites
Make sure you have your desired API keys:
- `OPENAI_API_KEY`
- `ANTHROPIC_API_KEY`
- `GEMINI_API_KEY`

### Installation
Ensure you install the core NLP libraries and Streamlit:
```bash
pip install openai anthropic google-generativeai streamlit
```

### Running the App
Run the interactive Streamlit Dashboard:
```bash
streamlit run app.py
```

Run the headless CLI Pipeline:
```bash
python agent.py
```
