# 🎭 Voice-Verify AI (Brand Voice Consistency)

Perform deep neural audits of content resonance against established brand guidelines with surgical alignment fixes.

## 🚀 Overview
**Voice-Verify AI v2.0** is an enterprise-grade linguistic integrity engine. It goes beyond simple spell-check, using multi-model intelligence to identify **vibe mismatches**, evaluate **trait alignment**, and provide **surgical edits** that preserve brand authority while fixing stylistic deviations.

## ⚡ Key Features
- **Phased Alignment Scoring**: Provides a high-fidelity assessment of how well your content matches the intended brand DNA.
- **Surgical Deviation Fixes**: Identifies problematic segments (Slang, Hype, Casual) and suggests professional alternatives in real-time.
- **Trait-by-Trait Diagnostics**: Breaks down the audit into specific guidelines like "Confidence", "Professionalism", or "Approachability".
- **Multi-Provider Architecture**: Native support for selecting your AI brain (OpenAI, Gemini, Claude, DeepSeek, or Groq).
- **Sentiment Resonance Check**: Analyzes the emotional "frequency" of the writing to ensure it hits the right note with high-value audiences.

## 🛠️ Tech Stack
- **Frontend**: Streamlit (Premium Creative UI with Glow-Pulse Score)
- **Intelligence**: LiteLLM (Multi-model support)
- **Data Protocols**: JSON for audit logic, TXT for linguistic brief export.

## 📂 Structure
- `agent.py`: Core linguistic engine and multi-provider CLI wrapper.
- `app.py`: Premium animated Streamlit dashboard with surgical review.
- `guidelines.txt`: Default brand voice traits for testing.
- `content.txt`: Sample content for audit simulation.
- `requirements.txt`: Project dependencies.

## 🚀 Quick Launch

### 1. CLI Usage
```bash
python agent.py
```

### 2. Dashboard Usage
```bash
streamlit run app.py
```

## 📊 Strategic Output
The agent outputs a structured JSON analysis including:
- **Alignment Score**: A numerical representation of brand resonance.
- **Diagnostic Verdict**: Critical/Moderate/Excellent identification.
- **Surgery Map**: Specific segments with problem definitions and proposed replacements.
